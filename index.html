Metrics Explanation

Edit Similarity
Length-Normalized Edit Similarity: $$S = 1 - \frac{\text{ED}(\hat{y}, y^*)}{{\max(|\hat{y}|, |y^*|)}}$$ where ED is the Levenshtein edit distance. Ranges from 0 to 1, with higher values indicating better performance.
Granularities
Four levels of code generation complexity: Class (complete classes), Function (entire functions), Block (code blocks), and Line (single lines). Difficulty decreases from Class to Line.
Languages
18 programming languages supported, divided into full-granularity (11 languages) and partial-granularity (7 languages) groups.
Dataset
1,286 human-verified test cases for evaluation, plus 17,000+ training tasks in M¬≤G-Eval-Instruct dataset.
